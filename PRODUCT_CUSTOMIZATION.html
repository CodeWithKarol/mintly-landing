2. Global Styles & Branding

Customize the core look and feel of the landing page by modifying CSS variables in style.css.

Color Palette

Locate the :root selector at the top of style.css to change the brand colors. 

:root {
    /* Primary Brand Colors */
    --color-teal: #14b8a6;       /* Main accent color */
    --color-teal-dark: #0d9488;  /* Hover states */
    
    /* Text & Background Colors */
    --color-navy: #001f3f;       /* Headings */
    --color-coral: #ff6b6b;      /* Secondary accents */
    --color-white: #ffffff;      /* Backgrounds */
    --color-light-bg: #f8fafb;   /* Section backgrounds */
}

Typography

The project uses the "Inter" font family from Google Fonts. To change it:

  1. Update the Google Fonts <link> in the <head> of index.html.
  2. Update the --font-family variable in style.css.

3. Header & Navigation

The header contains the logo, navigation links, and CTA buttons. It becomes sticky on scroll.

Customizing the Logo

The logo is an SVG inside the .logo anchor tag. You can replace the SVG code or use an <img> tag. 

<!-- In index.html -->
<a href="#" class="logo">
    <!-- Replace SVG below with your logo -->
    <svg class="logo-icon" ...>...</svg>
    <span class="logo-text">Mintly</span>
</a>

Navigation Links

Edit the links in the .nav-links div. Ensure the href matches the section IDs (e.g., #features).

Note: Remember to update both the Desktop Navigation (.nav-links) and Mobile Navigation (.mobile-nav-content) sections in index.html.

4. Hero Section

The first section users see. It includes the main value proposition and trust indicators.

Copywriting Guide

  • Badge: .hero-badge - Short, punchy proof point (e.g., "#1 AI Finance App").
  • Headline: .hero-heading - Main benefit. Use <span class="text-gradient"> for color emphasis.
  • Subheading: .hero-subheading - Elaborate on the solution and key benefits.

Visuals

The hero image is located in .hero-image-container. Replace the src attribute of the img tag. 

<img 
    src="YOUR_IMAGE_URL" 
    alt="Description of image" 
    class="hero-main-image" 
/>

Floating Cards: The floating stats (Savings, Investment) are HTML elements .float-card overlaid on the image. You can edit their values directly in the HTML.

5. Features & Benefits

Located in the <section class="benefits" id="features">. It uses a Bento Grid layout.

Customizing Cards

There are two types of cards: .large-card and .small-card.

  • Icons: SVG code inside .icon-wrapper. Replace with any SVG icon (24x24px recommended).
  • Text: Edit the <h3> and <p> tags inside each card.
  • Images: Large cards have a .card-visual div containing an image.

6. How It Works

A step-by-step timeline guide located in <section class="how-it-works">.

Structure

Each step is a .process-step div. Every second step has the class .reverse to alternate the layout.

  • Step Number: .step-badge (e.g., "Step 01").
  • Content: Edit the title, description, and the .check-list items.
  • Visual: Each step has a .visual-card with an image and a floating badge (e.g., "Connected", "Goal: $50k").

7. Pricing Section

Displays 3 pricing tiers. The toggle switch is visual-only in this template but can be hooked up to JS logic.

Editing Plans

Each plan is a .pricing-card. The middle card has the .popular class for emphasis.

  • Price: Update the number in .price-value. It has data attributes data-monthly and data-yearly which you can use for JavaScript toggle logic.
  • Features: Edit the <li> items in .pricing-features.
  • Links: Update the CTA button href (e.g., to your Stripe checkout).

9. FAQ Section

An accordion-style FAQ section. The interactivity is handled by script.js (implied, or uses simple CSS/HTML details/summary if not script-heavy, but here it uses buttons).

Adding Questions

Copy an existing .accordion-item block.

  • Question: Inside .accordion-trigger .accordion-title.
  • Answer: Inside .accordion-content .accordion-body.

The JavaScript for toggling these is likely generic, looking for .accordion-trigger clicks.

10. Footer & Newsletter

The footer includes links, newsletter signup, and copyright info.

Newsletter Form

The form has an onsubmit="handleNewsletter(event)" attribute. You will need to implement this function in script.js to connect to your email service provider.

Links

Organized in columns (.footer-col). Update the <a> tags to point to your actual pages.

11. Interactive Elements

Chat Widget

The floating chat widget is defined in .chat-widget-container.

  • Initial Message: Edit the text in .bot-message inside .chat-body.
  • Quick Actions: Edit the buttons in .quick-actions.
  • Logic: The toggleChat() function in script.js controls visibility.

Video Modal

The "Watch Demo" button in the Hero section opens a modal.

  • Video Source: Locate #video-modal at the bottom of index.html. Change the <source> src to your video file or URL.
  • Trigger: The button with id watch-demo-btn triggers the modal.
